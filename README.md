# Khushi Tirkey | Portfolio

A modern, high-performance personal portfolio built with vanilla HTML, CSS, and JavaScript.

## Features

- **Dark Mode Aesthetic** — Slate + Emerald color palette with high-contrast typography
- **Glassmorphism Header** — Blur effect that intensifies on scroll
- **Typing Effect** — Rotating taglines in the hero section
- **Project Filtering** — Filter projects by category (AI/ML, Full-Stack, Computer Vision)
- **Smooth Animations** — Scroll reveals, hover effects, gradient borders, and particle effects
- **Contact Form** — Privacy-first contact form (no phone number)
- **Responsive Design** — Fully responsive with mobile hamburger menu

## Tech Stack

- HTML5
- CSS3 (Custom properties, backdrop-filter, animations, gradients)
- Vanilla JavaScript (Intersection Observer, DOM manipulation)
